# Claude Code 开发指南

本项目使用 Claude Code 进行 AI 辅助开发。

## 开发规范

请阅读并严格遵循 `AGENTS.md` 中的所有开发规范和编码约定。

## 关键要点

- 本项目追求**架构简洁、代码优雅、主流最佳实践**
- 使用 **Go 1.22+** 现代特性，不考虑旧版本兼容性
- 遵循**语义优先**原则：代码应该"读起来就是它做的事"
- 使用**表格驱动测试** + **testify 断言**
- 重点关注**边界测试**和**错误路径覆盖**

## 测试要求

所有代码修改必须：
1. 通过 `golangci-lint run` 检查（零问题）
2. 通过 `go test ./...` 测试（100% 通过率）
3. 核心逻辑保持 70%+ 测试覆盖率

## 提交前检查清单

```bash
# 1. 格式化
goimports -w .

# 2. Lint 检查
golangci-lint run

# 3. 运行测试
go test -cover ./...
```
